# [Trivializer](https://trivializer-game.herokuapp.com/)
## Overview
Trivializer is a free to play trivia game where users can test their knowledge, track their scores, and work their way up the leaderboard!

## Technologies Used
* React
* Node.js
* postgreSQL
* Express.js
* Bootstrap
